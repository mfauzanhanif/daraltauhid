# Super App Dar Al Tauhid

## Deskripsi Proyek

*Super App Dar Al Tauhid* adalah platform digital terpusat yang dirancang untuk mendigitalisasi seluruh operasional Yayasan Dar Al Tauhid beserta lembaga-lembaga pendidikan di bawah naungannya.

Sistem ini dibangun menggunakan pendekatan *Hybrid Modular Monolith* dengan arsitektur *Multi-Domain (Path-Based Tenancy)*. Pendekatan ini memungkinkan satu basis kode (codebase) untuk menangani berbagai domain lembaga dan logika bisnis yang kompleks (Keuangan, Akademik, Kesiswaan) secara terisolasi, namun tetap efisien berjalan di lingkungan Shared Hosting.

Aplikasi ini dirancang future-proof dengan pemisahan modul yang tegas, mendukung pengembangan aplikasi mobile (Android/iOS) menggunakan wrapper native.

# TECH STACK

| Kategori          | Teknologi         | Versi / Catatan                                                        |
| ----------------- | ----------------- | ---------------------------------------------------------------------- |
| Core Framework    | Laravel 12        | PHP 8.2+. Memanfaatkan fitur terbaru, struktur skeleton yang bersih.   |
| Server Side Logic | Livewire 3 (Volt) | Functional API. Menulis logic PHP dan HTML dalam satu file .blade.php. |
| UI Framework      | Tailwind CSS v4   | Utility-first CSS. Versi 4 lebih cepat (Rust-based engine).            |
| Component Library | Flowbite          | Kumpulan komponen UI (Modal, Sidebar, Table) berbasis Tailwind.        |
| Interactivity     | Alpine.js         | Menangani state UI ringan (toggle, dropdown) di sisi klien.            |
| Database          | MySQL / MariaDB   | Penyimpanan data relasional terpusat.                                  |
| Build Tool        | Vite              | Bundling aset JS/CSS. (Build lokal -> Upload ke Hosting).              |

# Super App Dar Al Tauhid

## Deskripsi Proyek

**Super App Dar Al Tauhid** adalah platform digital terpusat yang dirancang untuk mendigitalisasi operasional Yayasan Dar Al Tauhid beserta lembaga-lembaga di bawah naungannya.

Sistem ini menggunakan arsitektur **Multi-Domain Monolith**, di mana satu basis kode (codebase) menangani berbagai domain dan subdomain lembaga untuk efisiensi sumber daya server (Shared Hosting).
